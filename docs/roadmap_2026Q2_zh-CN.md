# Roadmap（2026 Q2）

> 时间窗口：2026-02-16 ～ 2026-05-22（14 周）

## 1. 里程碑总览

```text
M1 激活与可诊断（W1-W4）
M2 留存核心能力（W5-W8）
M3 深度使用与发布治理（W9-W14）
```

## 2. 里程碑拆解

### M1：激活与可诊断（W1-W4）

交付项：
- [x] `changqiao doctor`
- [x] 工作区快照（启动恢复 + 退出持久化）
- [ ] 首次启动引导（关键快捷键 + 常见报错修复指引）
- [ ] Token 到期前提示
- [ ] 配置错误分级提示（可修复建议）

验收标准：
- 首次启动成功率 ≥ 80%
- 配置类 FAQ 占比下降 ≥ 30%

---

### M2：留存核心能力（W5-W8）

交付项：
- [x] 预警规则引擎骨架（本地 JSON + 命中评估 + 日志告警）
- [x] 工作区记忆（symbol、排序、分组、上次页面）
- [ ] 命令面板（统一动作入口）
- [ ] 失败兜底展示（stale-but-usable）

验收标准：
- 预警配置用户占活跃用户 ≥ 30%
- D7 留存较基线提升 ≥ 20%

---

### M3：深度使用与发布治理（W9-W14）

交付项：
- [ ] 多标的对比视图
- [ ] 复盘导出（Markdown / CSV）
- [ ] 发布分轨（stable / preview）
- [ ] 应用内反馈入口（自动生成 issue 模板）
- [ ] 回滚演练并固化 runbook

验收标准：
- Crash-free sessions ≥ 99.5%
- 发布后 24h 无 P0 事故
- 版本升级成功率 ≥ 95%

## 3. 优先级分层

```text
P0（必须）:
  doctor, 首次引导, 预警中心(最小版), 工作区记忆, 发布分轨

P1（应做）:
  多标的对比, 复盘导出, 自选分组增强

P2（可后置）:
  云同步, 插件系统, 模板市场
```

## 4. 发布节奏建议

1. 每两周一个小版本（0.8.x）
2. 每个版本只承载 1～2 个 P0
3. 周五执行 preflight + 冒烟 + 回滚演练
